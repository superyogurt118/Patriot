# PatriotOS 1.0 🇷🇺

**PatriotOS** — концепт-прототип российской операционной системы, разработанный в образовательных и демонстрационных целях. Проект представляет собой веб-эмулятор рабочего стола с полноценной оконной системой и предустановленными приложениями, демонстрирующий возможности отечественного ПО.

> \*«Самая главная операционная система в России через десять лет начинается с одной HTML-страницы сегодня.»\*
>
> \*«Своих не бросаем. Чужих не запускаем.»\*

\---

## ✨ Возможности

### 🖥️ Интерфейс

* Экран блокировки с анимированным входом
* Рабочий стол со сменяемыми обоями (флаг, герб, кремль, просторы, берёза)
* Кнопка **«Пуск»** с меню всех приложений
* Вертикальный док с иконками
* Панель даты/времени с индикатором праздников
* Полноценная оконная система с перетаскиванием
* Адаптивный дизайн для всех устройств

### 📦 Предустановленные приложения

|Приложение|Описание|
|-|-|
|**Калькулятор**|Математические вычисления|
|**Браузер**|Просмотр веб-страниц (только .ru, .рф)|
|**RuStore**|Магазин приложений|
|**Румузыка**|Музыкальный плеер с патриотическими плейлистами|
|**Рувидео**|Видеоплеер|
|**Файлы**|Файловый менеджер с виртуальной ФС|
|**Настройки**|Параметры системы|
|**Календарь**|Производственный календарь РФ|

### 🗣️ Диктор

* Три режима: при наведении, при клике, при выделении
* Поддержка русского и белорусского языков
* Озвучивание текста интерфейса

### 🔒 Безопасность и данные

* Все данные хранятся локально (localStorage)
* Валидация вводимых данных
* Ограничение на российские домены
* Сохранение настроек, файлов и установленных приложений

### 🌐 Локализация

* Полный перевод интерфейса на русский и белорусский языки
* Названия праздников на обоих языках
* Мгновенное переключение без перезагрузки (с обновлением страницы)
