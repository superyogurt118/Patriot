## 📋 О проекте

**PatriotOS** — концепт-прототип российской операционной системы, разработанный в образовательных и демонстрационных целях. Проект представляет собой веб-эмулятор рабочего стола с оконной системой и предустановленными приложениями, демонстрирующий возможности отечественного ПО.

> *«Самая главная операционная система в России через десять лет начинается с одной HTML-страницы сегодня.»*

---

## ✨ Возможности

### 🖥️ Рабочий стол
- Сменяемые обои (флаг РФ, герб РФ, берёза)
- Вертикальный док с системными приложениями
- Панель даты и времени с производственным календарём РФ
- Кнопка "Пуск" с меню приложений
- Область для иконок приложений

### 📦 Предустановленные приложения
| Приложение | Назначение |
|------------|------------|
| **Счётчик** | Калькулятор |
| **Браузер** | Просмотр веб-страниц |
| **RuStore** | Установка приложений |
| **Настройки** | Параметры системы |
| **Документы** | Файловый менеджер |
| **Календарь** | Производственный календарь РФ |

### 🌐 Браузер
- Встроенный браузер на основе iframe
- Навигация: назад/вперёд/обновить
- Домашняя страница с информацией

### 📁 Файловая система
- Виртуальная файловая система
- Создание/удаление папок и файлов
- Поддержка текстовых файлов, изображений, аудио

### 🗓️ Производственный календарь
- Официальные праздники РФ
- Отметка текущего дня
- Навигация по месяцам
- Список праздников в текущем месяце

### 📱 Мобильная поддержка
- Адаптивный дизайн
- Поддержка сенсорного управления
- Оптимизация под все устройства
