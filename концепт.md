## **🎯 О ПРОЕКТЕ**

**PatriotOS** — это **концепт-прототип** российской операционной системы, демонстрирующий возможности отечественного ПО. Проект представляет собой веб-эмулятор рабочего стола с полноценной оконной системой и набором приложений.

> *«Самая главная операционная система в России через десять лет начинается с одной HTML-страницы сегодня.»*

---

## **✅ ЧТО УЖЕ СДЕЛАНО**

### **🖥️ Интерфейс**
- Экран блокировки с часами и кнопкой входа
- Рабочий стол со сменяемыми обоями (флаг, герб, берёза)
- Кнопка **«Пуск»** с модальным меню приложений
- Вертикальный док с системными приложениями
- Панель даты/времени с индикатором праздников
- Полноценная оконная система (открыть/закрыть/свернуть/развернуть/перетаскивание)

### **📱 Мобильная поддержка**
- Адаптивный дизайн для всех устройств
- Touch-события для всех элементов
- Запрет масштабирования
- Плавная прокрутка на iOS

### **📦 Предустановленные приложения**
| Приложение | Назначение |
|------------|------------|
| **Счётчик** | Калькулятор |
| **Браузер** | Просмотр веб-страниц |
| **RuStore** | Магазин приложений |
| **Настройки** | Параметры системы |
| **Документы** | Файловый менеджер |
| **Календарь** | Производственный календарь РФ |

### **🌐 Браузер**
- Навигация (назад/вперёд/обновить)
- Адресная строка
- Домашняя страница
- Белый список российских доменов
- Есть поиск, но в перспективах временно удалить

### **📁 Документы (Файловый менеджер)**
- Виртуальная файловая система
- Папки: Документы, Изображения, Музыка
- Создание/удаление/переименование
- Открытие `.txt`, `.mp3`, `.png/.jpg`

### **📦 RuStore**
- Установка предустановленных приложений на рабочий стол
- Добавление пользовательских приложений
- Сохранение в localStorage
- Только российские домены

### **📅 Календарь**
- Производственный календарь РФ 2026
- Отметка официальных праздников ⭐
- Выделение текущего дня
- Список праздников в месяце
- Навигация по месяцам

### **🔒 Безопасность**
- Блокировка F12 и инструментов разработчика
- Отключение контекстного меню
- Все данные хранятся локально

## **📋 ТЕХНИЧЕСКИЕ ДЕТАЛИ**

### **Технологии**
- HTML5, CSS3, Vanilla JavaScript
- localStorage для хранения
- Flexbox/Grid для адаптива
- Без фреймворков

### **Структура**
```
PatriotOS/
├── index.html
├── wallpapers/    # Обои
├── sounds/        # Звуки
└── images/        # Иконки приложений
```

### **Запуск**
1. Скачать все файлы
2. Открыть `index.html` в браузере
3. Никакой установки не требуется
